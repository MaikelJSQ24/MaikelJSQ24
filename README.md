### 🚌 Sistema de Rutas de Transporte Público
> **Aplicación de Escritorio · Estructuras de Datos y Algoritmos**

Sistema para crear, editar y visualizar rutas de transporte público. Implementa algoritmos clásicos de grafos para calcular rutas óptimas entre paradas.

| Algoritmo | Uso |
|-----------|-----|
| 🔵 Dijkstra | Ruta más corta entre dos paradas |
| 🟣 Floyd-Warshall | Distancias entre todos los nodos |
| 🟢 Prim | Árbol de expansión mínima de la red |
| 🔄 BFS (A lo ancho) | Exploración por niveles de paradas |
| 📉 DFS (A lo profundo) | Exploración en profundidad de rutas |
